# Музыкальный дневник

## Описание проекта
Музыкальный дневник — это программа с графическим интерфейсом на Python и Tkinter, предназначенная для сохранения информации о прослушанных треках. Пользователь может ввести название трека, исполнителя, жанр, настроение, комментарий и поставить оценку по пятибалльной шкале.

Если трек получает оценку 5, он автоматически попадает в основной плейлист. Если оценка ниже 5, запись сохраняется в общий журнал музыкального дневника.

Программа выполняет проверку введённых данных, сохраняет записи в файл JSON, показывает статистику по добавленным трекам и ведёт логирование всех действий пользователя в отдельный файл.

## Возможности
- добавление новых треков;
- оценка треков по шкале от 1 до 5;
- автоматическое распределение записей;
- просмотр основного плейлиста;
- просмотр общего лога;
- просмотр статистики;
- сохранение данных в JSON;
- логирование действий в отдельный файл.

## Технологии
- Python
- Tkinter
- JSON
- logging
