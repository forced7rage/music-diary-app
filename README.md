# Музыкальный дневник от F7R ☊

``` Программа для сохранения и анализа прослушанных музыкальных треков, разработанная на Python ```



###  Описание проекта

> Музыкальный дневник — это приложение с графическим интерфейсом, предназначенное для ведения записей о прослушанных композициях.

### Практическая значимость программы:
Это удобное место, где сохраняется каждая понравившаяся песня: добавляете песню, ставите оценку, а программа распределит в нужное место. Любимые песни, с оценкой 5 всегда под рукой – в отдельном плейлисте, а остальные – в общий журнал. Все записи аккуратно хранятся, их удобно пересматривать и дополнять своими комментариями. Простой   способ не потерять ни одну важную для вас музыку!

### Особенность:

* оценка **5** → трек попадает в основной плейлист
* оценка **ниже 5** → запись попадает в общий журнал

Все данные сохраняются в файл **JSON**, а действия пользователя записываются в лог-файл.
